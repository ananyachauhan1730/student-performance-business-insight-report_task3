# Student Performance Business Insight Report

## AIML Internship - Week 1 Assignment

### Project Overview

This project analyzes student performance data to identify the key factors that influence academic outcomes. The analysis includes data cleaning, visualization, and business insight generation.

### Dataset Used

Student Performance Dataset

### Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn

### Tasks Performed

1. Data Cleaning
2. Missing Value Analysis
3. Duplicate Record Checking
4. Descriptive Statistics
5. Data Visualization
6. Correlation Analysis
7. Business Insight Report Generation

### Key Findings

* Higher attendance is associated with better final grades.
* Students who study more hours tend to achieve higher scores.
* Previous academic performance strongly influences final grades.
* Extracurricular activities show a moderate positive relationship with performance.
* No significant gender imbalance was observed in the dataset.

### Recommendations

* Encourage regular attendance.
* Promote consistent study habits.
* Provide additional support to low-performing students.
* Monitor students with declining attendance patterns.
